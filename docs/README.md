# CONECTA - Landing Page

Esta es la versión estática de la landing page de CONECTA para GitHub Pages.

## Cómo activar GitHub Pages

1. Ve a tu repositorio en GitHub: `https://github.com/Laummendoza/conecta-backend`
2. Click en **Settings** (Configuración)
3. En el menú lateral, click en **Pages**
4. En **Source**, selecciona la branch que quieres usar (ej: `main` o `claude/als-app-landing-page-011CUqFR6SMswk8KZ3Ts5qqL`)
5. En **Folder**, selecciona `/docs`
6. Click en **Save**
7. Espera unos minutos y tu sitio estará disponible en:
   `https://laummendoza.github.io/conecta-backend/`

## Vista previa local

Para ver la página localmente sin servidor, simplemente abre el archivo `index.html` en tu navegador.

## Archivos incluidos

- `index.html` - Landing page principal
- `logo.jpg` - Logo de CONECTA
- `README.md` - Este archivo
